# 🌸 Dia das Mães - Gerador de Homenagens

Um projeto feito em **HTML, CSS e JavaScript** para criar lindas homenagens de Dia das Mães diretamente pelo navegador ❤️

O usuário pode:
- escrever uma mensagem
- enviar uma foto
- escolher templates diferentes
- salvar tudo automaticamente
- exportar como imagem
- compartilhar nas redes sociais

---

✅ Instagram Style

O projeto foi desenvolvido pensando primeiro em celulares, para ficar perfeito em:
- Instagram Stories

- compartilhamentos mobile

# ▶️ Como Usar

1. Abra o projeto no navegador
2. Clique em **Editar**
3. Escolha:
   - nome
   - mensagem
   - foto
   - template
4. Clique em **Salvar**
5. Clique em **Salvar Imagem**
6. Compartilhe ❤️
